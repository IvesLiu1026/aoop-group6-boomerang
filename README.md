## Group 6 AOOP Project
This repo aims to make a clone of Boomerang Fu using pygame. We will put the skeleton of the game here.

- Objective
    - Movement    
        - WASD
        - Sprint
        - Flash
    - Weapons
        - hands
            - Auto attack      
        - boomerang     
            - Throw
    - Obstacle
    - UI Interaction(might be rough or just CLI)
